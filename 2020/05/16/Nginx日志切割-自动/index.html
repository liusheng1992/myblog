重启定时任务

常用定时任务命令
// 启动服务
service crond start
// 关闭服务
service crond stop
// 重启服务
service crond restart
// 重启载入配置
service crond reload
// 编辑任务
crontab -e
// 查看任务列表
crontab -l

定时任务表达式

时间范围 分 时 日 月 星期 年(可选)
取值范围 0-59 0-23 1-31 1-12 1-7 2019/2020/2021/…

常用表达式

  • 每分钟执行
  • 每日凌晨（每天晚上23.59）执行
  • 每日凌晨1点执行
